# start-new-tab-page
Это короче стартовая страница для браузера, можно создавать группы вкладок, ставить видео и плейлисты с ютаба на фон, можно сохранять конфигурацию свою и на дргуие компики переносить, работает с ооп, можно свои какие то приколы достаточно легко добавлять, а еще там редактор частиц крутой
